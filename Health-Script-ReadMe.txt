VDP-Health-Script Instructions:

1. Download the health.zip file
2. Place this file under /home/admin
3. Unzip the folder
# unzip health.zip

3. Give the extracted folder and files read and execute permissions recursively for all users.
# chmod -R a+r health
# chmod -R a+x health

4. change directory to health
# cd health

5. Run the ./health.script (Make sure you are 'admin' when you run this script)
